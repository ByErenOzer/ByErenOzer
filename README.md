## 🚀 Featured Projects

### 🏥 Real-Time Tuberculosis Bounding Box Detection & Localization
An advanced healthcare AI pipeline developed using YOLO26x and custom batch-level samplers to localise pulmonary tuberculosis lesions on chest X-ray scans.
*   **Key Features:** Automated coordination scaling bug correction (re-scaling VIA 1024 space to high-res raw images), Class-Aware Sampling (CAS), Repeat Factor Sampling (RFS) for extreme class imbalance.
*   **Results:** Reached **0.8829 F1-score** on Montgomery County dataset and **0.5662 F1-score** on VinDr dataset.
*   **Stack:** PyTorch, YOLOv8/v11/v26, OpenCV, Hugging Face Hub, Pandas, Scikit-Learn.
*   **Repositories:** Code: [ByErenOzer/Tuberkloz_Tespiti](https://github.com/ByErenOzer/Tuberkloz_Tespiti) | Dataset: [erenozer/Tuberculosis_CXR_Datasets](https://huggingface.co/datasets/erenozer/Tuberculosis_CXR_Datasets)
